# Markmono Font

This font is derived from the [Monocode](https://github.com/aaronmbos/monocode) font, which in turn is a customized version of [Iosevka](https://typeof.net/Iosevka/); therefore, this is also a customized version of Iosevka.

The intention of this version is to offer a version with ligatures and [Nerdfont](https://www.nerdfonts.com/) symbols.

The last release was built with Iosevka version 34.3.0.
